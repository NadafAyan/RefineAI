# Project Name: RefineAI (PromptForge)
# Stack: Next.js 15 (App Router), TypeScript, Tailwind, Shadcn UI, Lucide Icons, Firebase Auth.
# Goal: A SaaS platform that helps naive users refine their AI prompts using a step-by-step wizard.

## Design Vibe
- "Cyberpunk Professional": Dark mode by default, neon accents (purple/blue), glassmorphism cards.
- Layout: Large typography, clean spacing, "Mission Control" aesthetic.

## Core Features
1. Landing Page: Dynamic, contains "Documentation" tabs explaining how to use the tool.
2. Auth: Google Sign-In via Firebase (Pop-up).
3. Dashboard: A "Wizard" form (Step 1: Objective, Step 2: Persona, Step 3: Constraints).
4. Output: Side-by-side comparison of "Naive Prompt" vs "Refined Prompt".
